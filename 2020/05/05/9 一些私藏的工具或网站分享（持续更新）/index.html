一些私藏的工具或网站分享（持续更新）

此篇博客长期更新一些方便的网站或工具。

网站篇

#【瓜子解析】（http://vip.guazivip.com/）免费 无注册

爱奇艺，腾讯视频没有vip，有它随便看。使用教程：http://vip.guazivip.com/jiaocheng.html
类似的网站有很多，如果出现无法解析的视频时可以尝试以下网站解析：【618G】、【石头解析】、【玩的嗨】、【最惠买】、【http://v.ctrlqq.com/】、【】

#【公众号 Markdown 编辑器】（https://prod.zkqiang.cn/wxeditor/）免费 无注册

markdown 语法写的文章，微信公众号不支持，Markdown-here 解析的排版丑陋，那就试试它吧！（目前本人在用）

#【在线图片压缩】（https://en.m.fontke.com/tool/compressimage/）免费 可选注册

试过很多款，觉得这款ok，没有强制注册登录，可以多张图片一起压缩打包下载。

#【本地图片压缩】（https://tool.tanpok.com/）免费 无注册

支持批量图片压缩和加水印，全程免费。

#【unsplash】（https://unsplash.com）免费 可商用 可选注册

试过很多图片网站，很多需要钱或积分，但这款统统不需要。站上图片基本属于摄影图片，被图片拍摄者分享。网站服务器在国外，访问网速因人而异。

#【pexels】（https://www.pexels.com/）免费 可商用 可选注册

和unsplash类似，但是这个网站比unsplash快一些。图片质量还行。下载时可选分辨率。

#【随心听歌】（http://ziliao6.com/fm/）免费 无注册

不知道听什么，不希望页面复杂，那就听它吧！随机播放，没有歌单，没有搜索，只有上一曲下一曲和暂停。让音乐简单点。

#【毒鸡汤生成器】（https://du.shadiao.app/）免费 无注册

每天一碗毒鸡汤。
其他类似：【朋友圈文案生成器】（https://pyq.shadiao.app） 【骂人宝典】（https://nmsl.shadiao.app）

#【谷歌插件下载】（https://chromecj.com） 免费 可选注册

下载插件需要先关注一个公众号并获取6位验证码。 网站广告偏多

#【优品PPT】（http://www.ypppt.com）免费 无注册

大量PPT模板免费下载，找PPT模板，首推这个网站。

#【WPS智能PPT】（http://jm.wps.cn）免费 需登录

需要登录，可以根据模板填写页面所需内容，自动排版，免费下载。总的来说，模板比较少，排版也一般（字体大小，颜色这些还是需要自己调才好看），如果制作PPT很紧急，可以采取这种办法；否则，建议找个模板自己慢慢做。

#【嗖嗖搜】（http://www.bewindoweb.com/dwg.php）免费 无注册

包含一些常用网站汇总，大部分和程序员相关。

#【创客贴】（https://www.chuangkit.com）部分免费 需登录

可以设计一些海报，新媒体配图，头像等，基本拖动素材就行，但是很多素材需要收费，如果是设计师，一般会开会员，单个素材计费很贵。本人公众号头像便是在这设计。

#【PanDownload】（http://pandownload.com/）免费 百度网盘登录

百度网盘没有会员会限速？试试这个。之前 PanDownload 速度变慢了，现在我测试了新更新的2.2.2版本，速度能到5-6 M/s。这种免费的软件偶尔不稳定。

#【ENFI下载器】（http://enfi.cloud/）收费 无登录

这款下载器无需登录，复制别人分享的百度网盘链接即可。最大的特点是按需下载，最低2块钱15G。速度2-5M/s，用了很久，比较稳定。

#【SetupVPN】（谷歌浏览器插件） 免费 需注册

一把免费的浏览器插件梯子，这些技术一般不稳定，但这款我感觉相对稳定点。一直在用。可自行在谷歌商店搜索下载。

#【Stylish】（谷歌浏览器插件） 免费 无注册

#【Video Speed Controller】（谷歌浏览器插件） 免费 无注册

无论网站是否提供倍速播放，都可对网站视频最高16倍速播放。（之前淘宝教育的视频无法倍速，可以用它）

#【在线简历制作】（https://www.polebrief.com/）免费 无登录

无需登录，即可完成极简简历的制作和下载。

#【Magi】（https://magi.com/）免费 无登录

基于机器学习的信息抽取和检索系统，它能将任何领域的自然语言文本中的知识提取成结构化的数据，通过终身学习持续聚合和纠错，进而为人类用户和其他人工智能提供可解析、可检索、可溯源的知识体系。你得区别于百度，它会通过学习来定义你所搜索的东西。去玩玩吧。

#【查询猫】（https://www.cxmao.cn/）免费 无登录

如果你怀疑某个qq，微信或手机号是骗子，可以在此平台查证。

#【一个工具箱】（http://www.atoolbox.net）免费 无登录

一个聚合多种在线工具的网站。各种意想不到的工具都在这里。

#【图片无损放大】（https://bigjpg.com）免费 可选注册

使用最新人工智能深度学习技术，将噪点和锯齿的部分进行补充，实现图片的无损放大。

#【新媒体管家】（https://xmt.cn）（浏览器插件）免费 关注公众号 手机号绑定

这个插件里面有很多样式可以免费用于公账号文章的排版。类似平台：【壹伴】

#【石墨文档】（https://xmt.cn） 免费 需登录

可实现多人同时在同一文档及表格上进行编辑和实时讨论，还可一键还原到任一历史版本。

#【墨刀】（https://modao.cc） 可选付费 需登录

专业、可靠、简洁、易懂的在线产品原型设计与协作平台。免费版只能创建3个项目，企业版25元每月。

#【Color Hunt】（https://colorhunt.co） 免费 无注册

提供大量漂亮的颜色搭配方案，可用于网站或软件的一些设计上。

#【蛙蛙在线工具】（https://www.iamwawa.cn） 免费 无注册

提供一些常用工具集合。

#【nicetool】（http://www.nicetool.net） 部分收费 无注册

提供一些常用工具集合。部分工具收费

#【iconfont】（https://www.iconfont.cn） 免费 可选登录

阿里巴巴字体图标库，免费下载使用。

#【echarts】（https://www.echartsjs.com/） 免费 无注册

一个纯 Javascript 的图表库，免费使用，官网提供大量官方实例可以copy。

#【全历史】（https://www.allhistory.com/） 免费 可选登录

一个不可多得的高质量精品历史知识网站，主要的特点就是以时空关系来呈现历史知识。（也有app）我觉得最好玩的功能就是八竿子打不着的人物之间的关系也能找出来。
